Установка:
---------

В папке tmp нужно создать папку templates_c:

 ```sh
 mkdir tmp/template_c
 ```

 В папке tmp нужно создать папку session:

  ```sh
  mkdir tmp/template_c
  ```

Установить права на запись для папки tmp и ее вложений:

 ```sh
 chmod 777 -R tmp/
 ```
 
Дамп базы даееых mysql лежит в папке db
